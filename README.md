# legacy-pvp
mcpe 1.6 vanilla texturepack for 1.21 with pvp mods, like custom skybox, short swords, lowfire, no particles, no clouds, 60fps fire, "Ubuntu Mono Bold" font

# compatible with new items
there are no black and purple blocks and items
